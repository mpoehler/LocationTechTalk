Datencontainer

  • Jedes Chart benutzt DataTable als Behälter für Daten
  • Format und Anzahl der Spalten abhängig vom Charttyp

Ein DataTable kann auf vier Arten erzeugt werden:

  1. DataTable + addColumn() / addRows() / addRow() / setCell()
  2. arrayToDataTable
  3. JavaScript Literal Initializer
  4. DataSource Query (Google Chart Datasource)
